# APJ Abdul Kalam Tribute Website

A simple and responsive tribute website dedicated to **Dr. A. P. J. Abdul Kalam**,  
the Missile Man of India and former President of India.

This project is built using **HTML and CSS** and focuses on clean design and responsiveness.

---

## Live Demo

 https://bhupendrakushwaha0108.github.io/APJ-Tribute-webste/

---

## Features

- Responsive design (works on mobile & desktop)
- Clean and minimal UI
- Tribute section with image and biography
- Beginner-friendly project structure

---

## Technologies Used

- HTML5  
- CSS3  
- Git & GitHub  
- GitHub Pages (for deployment)
